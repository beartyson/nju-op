#### 看个例子

```c
/**
 * 工具函数
 * 1. 输出指针指向的地址
 * 2. 将指针指向的内存解读为 long，并以 16 进制 16 位格式输出
 */
void printPtr(void * ptr) {
	printf("ptr = %p, *ptr = %016lx\n", ptr, *(long *)ptr);
}
```

```c
int main() {
	printPtr(main);
	printPtr(&main);
	
	printf("-----------------------------\n");

	int arr[10];
	printPtr(arr);
	printPtr(&arr);

	printf("-----------------------------\n");

	char const * arr2[10];
	printPtr(arr2);
	printPtr(&arr2);
}
```

> 结论：C 语言中所有东西都可以看成指针 ！并且一切皆可取地址 ！
