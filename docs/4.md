#### 工具函数

```c
/**
 * 1. 输出指针变量中保存的地址（即：指针指向的地址）
 * 2. 将指针指向的内存区域以 long 来解读，并以 16 进制 16 位的格式进行输出
 */
void _printPtr(void * ptr) {
	printf("ptr = %p, *ptr = %016lx\n", ptr, *(long *)ptr);
}
```

#### C 是由（内存和指针）组成的，一切均可取地址

```c
int main() {
  int arr[10];
  _printPtr(arr);
  _printPtr(&arr);
  _printPtr(&arr[0]);
}
```

```c
int main() {
  _printPtr(main);
  _printPtr(&main);
}
```
