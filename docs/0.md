#### The Shell Scripting Language

![image-20220405195636923](https://aliyun-oss-lpj.oss-cn-qingdao.aliyuncs.com/images/by-picgo/image-20220405195636923.png)

```bash
#!/bin/sh

a=1
b=2

if [ $a != $b ]
  then
    echo "a is not equal to b"
fi

if [ $a > $b ]
  then
    echo "no"
fi
```
