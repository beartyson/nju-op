`Z 表示函数的返回值类型为 int`、`3 表示 bar 字符串的长度`、`v 表示函数无参数`

- C++ 的这种对中间代码的函数名的命名规范

- 使得在 C++ 源码中，即使函数重名，但只要（参数不同 || 返回值类型不同），就能确保在中间代码中，函数名唯一

- 所以 C++ 就支持了函数重载
